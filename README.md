# receipts-analyzer
## Developer guide
### Requirements
- PostgreSQL
- .NET Core SDK 2.1
- npm

### Steps

1. Restore db from dump `\api\receipts-server\db\dump` 
1. Open pgAdmin
1. Create database
1. Click `Restore...`
1. Select file by clicking `...` at Filename field
1. Select tar file
https://i.imgur.com/uY15710.png
1. Click Restore
